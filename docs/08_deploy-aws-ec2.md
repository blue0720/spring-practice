# 08. AWS EC2デプロイ

対応フェーズ: フェーズ7

## 概要
AWS EC2へ手動デプロイし、動作確認を行う。

## Todo

- [ ] ドメイン取得の要否を検討
- [ ] EC2上にMySQLを構築(または将来的なRDS移行を見据えた構成にする)
- [ ] `./mvnw.cmd clean package` でjarをビルド
- [ ] jarをEC2に配置し `java -jar` で起動確認
- [ ] 必要に応じてsystemd化(再起動時の自動起動)
- [ ] 本番相当の`DB_PASSWORD`等の環境変数・シークレット管理方針を決定
- [ ] 本番環境での動作確認(会員登録・ログイン・ゲームプレイ・管理画面)
