# 04. ゲーム画面へのMVPロジック移植

対応フェーズ: フェーズ3

## 概要
既存の vanilla-JS MVP(`memory_merchant_mvp.html`)のゲームロジックを、`/game` 画面(Thymeleafページ構成)へ移植する。
交渉パズルのコアロジック(ゲージ計算・相性判定)はサーバー側に移さず、クライアントJSのまま維持する。
